## 🚀 What's New in v1.0.0

Welcome to the initial release of **BerryWave EDI Inspector**! This tool is designed to streamline the inspection and validation of EDI data files.

### ✨ Key Features of Community Edition
* **View EDI file:** X12 and EDIFACT
* **Locality:** EDI content remains on your machine. No WiFi needed.
* **EDI Samples:** Provided as examples
* **Color-highlighting:** Delimiters, separators, terminators detected automatically
* **Overview:** Summary of EDI standard, version, syntax details, and navigation to key segments
* **Structural validation and repair:** Reports enveloping errors.
Allows correction, re-inspection, and saving.

### 🐛 Bug Fixes & Improvements
* **Installation:** via zip file
* **Port selection:** By default, the application selects an unused port.
A specific port can be configured in application.yml.
* **Launch script:** edi-inspector.sh for Mac, Linux.

---

## 📦 Installation & Usage
1. Download the attached `edi-inspector.zip` file below.
2. Extract the contents to your preferred directory.
3. Run the launch script to start inspecting your EDI files.
